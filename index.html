How you create a shapely Point object in Python:


							import shapely
							pnt1 = shapely.Point([8.68089, 47.21776])

Your task: Create a point object of your current location.
Tipp: use openstreetmap.org to query your coordinates (right click ‚Üí show address).

  • Demo this in jupyter lab.
  • point out, that the working directory is central (especially when starting jupyter lab desktop)

Note:

  • The order of the coordinates on openstreetmap.org (OSM) is Latitude, Longitude
  • However, shapely expects the order to be Longitude, Latitude (first x, then y)
  • The Coordinate Reference System (CRS) of the coordinates displayed on OSM is WGS84 (EPSG: 4326)
  • However, we cannot assign a CRS to a shapely object
  • Querying a shapely object: 
    
									type(pnt1)   # returns the type of the object
									pnt1         # returns a little image of the object
									print(pnt1)  # prints a WKT representation of the object

We can now create a GeoSeries from our shapely objects:


							import geopandas as gpd

							places = gpd.GeoSeries([work, restaurant, swimmspot])

Note:

  • GeoSeries (and GeoDataFrames) have two handy methods:
    • .plot() returns a static matplotlib plot
    • .explore() provides an interactive map
  • Since we have not provided a Coordinate Reference System ‚Üí no basemap
  • The function gpd.GeoSeries() accepts CRS Information (using the EPSG Code) 
    
									places = gpd.GeoSeries([work, restaurant, swimmspot], crs = 4326)
  • A GeoSeries is a list of shapely objects with no attribute information.
  • To add attribute information, we need a GeoDataFrame:

							places_gdf = gpd.GeoDataFrame(geometry = places, crs = 4326)

							places_gdf["names"] = ["Work", "Restaurant", "Swimmspot"]

							places_gdf # note the difference to the GeoSeries

Importing vector data

  • Typically, vector data is not created by hand, but imported
  • There are numerous vector data formats, the most infamous one is shapfile
  • there are better alternatives: Geopackage for smallish datasets
  • geoparquet / geoarrow for larger datasets
